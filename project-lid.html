📌 About the project

This project was developed as part of my Introduction to Speech Processing course. The main goal was to build a system that can automatically recognize the spoken language from short audio recordings.

Working with real audio data helped me better understand how speech processing systems are trained and evaluated — and how much fine-tuning goes into getting a model to perform well.

📂 Dataset

I used an open-source dataset from Kaggle containing short speech samples from multiple languages. Using real audio data gave me hands-on experience with the messy, unpredictable nature of real-world speech — no two recordings are the same.

✨ Methods implemented

1

MFCC — Mel-Frequency Cepstral Coefficients

I started with a classical approach using MFCC features. This was the first stage where I learned how language identification works. The code was easier to build, but results were limited compared to deep learning methods.

Classical method
2

CNN — Convolutional Neural Network

I then moved to a modern CNN approach which produced significantly better performance. The most challenging part was improving accuracy — I had to carefully analyze errors in training, adjust parameters, and improve the model step by step.

Deep learning

📊 Results

The CNN model outperformed the MFCC baseline across all evaluation metrics. It was trained for 10 epochs with the best-performing checkpoint saved during training.
