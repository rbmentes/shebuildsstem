Week 4

Neural Networks From Scratch

Spring 2026 · Beginner-friendly · The Architecture Behind Deep Learning

Weeks 1–3 covered what ML is, how models learn, and decision trees. Now we reach the most important topic in modern AI: neural networks. Everything from image recognition to ChatGPT is built on this foundation. This week we build one from scratch — concept by concept.

🧬 Where the idea comes from

Neural networks were inspired by the human brain. Your brain contains around 86 billion neurons — each one receives signals, processes them, and passes a signal forward. Artificial neural networks mimic this structure with mathematics instead of biology.

⚡ The single neuron — the basic building block

A single artificial neuron takes several inputs, multiplies each by a weight (how important that input is), adds them all together with a bias (a fixed offset), then passes the result through an activation function to produce an output.

A single artificial neuron

🔢 The math inside one neuron

Every neuron computes the exact same two operations — a weighted sum, then an activation:

Step 1 — weighted sum z = (w₁×x₁) + (w₂×x₂) + (w₃×x₃) + b
Step 2 — activation function output = f(z) ← e.g. ReLU, Sigmoid, Tanh
Worked example — one neuron, three inputs
Inputs x₁=0.5 , x₂=0.8 , x₃=0.2
Weights w₁=0.4 , w₂=0.6 , w₃=0.3
Bias b = 0.1
z = (0.5×0.4) + (0.8×0.6) + (0.2×0.3) + 0.1
z = 0.2 + 0.48 + 0.06 + 0.1 = 0.84
ReLU(0.84) = 0.84 ✓ (positive, passes through)

🏗️ From one neuron to a full network

A neural network is just many neurons organised into layers. Each layer's output becomes the next layer's input — this is called a feedforward or fully connected network.

📥

Input layer

Receives the raw data. One neuron per feature. No computation — just passes values forward.

🧠

Hidden layer(s)

Where learning happens. Can have many layers — each one learns more abstract features than the last.

📤

Output layer

Produces the final prediction. One neuron for regression, one per class for classification.

Fully connected neural network — 3 inputs → 4 hidden → 2 outputs

➡️ The forward pass — data flowing through

When we feed data into the network, it flows layer by layer from input to output. This is called the forward pass:

1

Input enters the network

Raw features (pixel values, MFCC coefficients, sensor readings) are loaded into the input layer neurons.

2

Each hidden neuron computes z = Wx + b

Every neuron in the hidden layer takes all outputs from the previous layer, multiplies by its weights, adds its bias, and applies activation.

3

Activations pass to the next layer

The activated outputs become the inputs to the next layer. This repeats for every hidden layer.

4

Output layer produces the prediction

The final layer produces the prediction — a number for regression, or probabilities per class for classification.

🐍 Building it in PyTorch

Here is a complete neural network in PyTorch — the same architecture as the diagram above:

Python — neural network in PyTorch import torch import torch.nn as nn # Define the network architecture class SimpleNN(nn.Module): def __init__(self): super().__init__() # Input(3) → Hidden(4) → Output(2) self.layer1 = nn.Linear(3, 4) # 3 inputs → 4 hidden neurons self.layer2 = nn.Linear(4, 2) # 4 hidden → 2 outputs self.relu = nn.ReLU() def forward(self, x): x = self.relu(self.layer1(x)) # hidden layer + activation x = self.layer2(x) # output layer (no activation) return x # Create the model model = SimpleNN() print(model) # Run a forward pass with fake data x = torch.tensor([[0.5, 0.8, 0.2]]) # one sample, 3 features output = model(x) print("Output:", output) # 2 values — one per output neuron # Count parameters total = sum(p.numel() for p in model.parameters()) print(f"Total parameters: {total}") # should print 26 (20 weights + 6 biases)

💡 Why deeper networks are more powerful

Layer 1 learns edges

The first hidden layer detects simple patterns — sharp transitions, low-level features in the data.

Layer 2 combines edges

The second layer combines simple patterns into more complex structures — shapes, phonemes, trends.

Layer 3 learns concepts

Deeper layers detect high-level abstract concepts — "face", "spoken word", "gesture pattern".

Output makes decision

The final layer uses all learned representations to make a confident classification or prediction.

💡 What I learned this week

A neural network is not magic — it's just many neurons stacked together, each doing the same simple operation: multiply inputs by weights, add a bias, apply activation. What makes it powerful is depth — each layer builds on the previous one, learning increasingly abstract representations. This is the foundation of every modern AI system from image classifiers to language models.
